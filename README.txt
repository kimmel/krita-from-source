1. Introduction 
---------------

Download and compile krita from source. To install the needed Perl modules
run the following command as root: 
    cpan IO::CaptureOutput File::Which File::Find::Rule

./krita_from_source.pl --help to see options


2. Requirements
---------------
Perl version 5.12 or later
CPAN Modules: IO::CaptureOutput File::Which File::Find::Rule
GNU wget



3. Reporting bugs
